## Terminal

---

Symlink
```bash
ln -s <target> <name-of-link>
```

RSync
```bash
rsync -ah --progress <from> <to>
```

List Infos of CPU
```shell
lscpu
```

List Size of Folders:
```shell
du -sh *
```

Check size of specific folder:
```bash
du -hs <folder>
```

Check how long PC is up:
```bash
uptime
```

Get number of files in dir and subdirs:
```bash
find . -type f | wc -l
```

Check Disc Usage
```bash
df -h
```
