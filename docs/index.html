MacOS Setup

How I set up fresh Machine

This shows how I set up my system


			  git status
			  git push

Customize your computer's name


          sudo scutil --set HostName <new-name> # this sets a new hostname

Make sublime text accessable via terminal


          ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

Toggle hidden files in Finder


          Command” + “Shift” + “.”

Setup SSH Configuration


          mkdir ~/.ssh # Creates a new hidden directory in your home folder
        

          subl ~/.ssh/config # Creates/Modifies the ssh configuration file

Key-Pair Authentication between Machines


          ssh-keygen # this generates a new pair of keys

copy the key to all your ssh remotes aliased <ssh machine> in your .ssh/config


          ssh-copy-id <ssh machine> # this generates a new pair of keys

Terminal

How to change the color scheme

Generate a new theme <mytheme> with with:


          subl ~/oh-my-zsh/themes/<mytheme>.zsh-theme

and insert the following text


          PROMPT="%{$fg_bold[green]%}%m "
          PROMPT+="%(?:%{$fg_bold[green]%}➜ :%{$fg_bold[red]%}➜ )"
          PROMPT+=' %{$fg[cyan]%}%c%{$reset_color%} $(git_prompt_info)'

          ZSH_THEME_GIT_PROMPT_PREFIX="%{$fg_bold[blue]%}git:(%{$fg[red]%}"
          ZSH_THEME_GIT_PROMPT_SUFFIX="%{$reset_color%} "
          ZSH_THEME_GIT_PROMPT_DIRTY="%{$fg[blue]%}) %{$fg[yellow]%}✗"
          ZSH_THEME_GIT_PROMPT_CLEAN="%{$fg[blue]%})"

Then open the following file with:


          subl ~/.zshrc

replace the following line


          ZSH_THEME="robbyrussell"

with your new theme


          ZSH_THEME="<mytheme>"

Allow a specific user to reboot machine

open the configuration file

sudo visudo

and add the following line


          <user> ALL=NOPASSWD:/sbin/reboot
